# 🚀 Smart Job Matcher

A powerful **AI-based job search assistant** that helps freshers and job seekers find relevant jobs, analyze their resumes, and improve their chances of getting shortlisted.

---

## 🔥 Features

- 🔍 Search jobs by **role and location**
- 🌍 Works for **any city or country**
- 📊 Filter jobs based on **experience level**
- 📄 Upload resume (PDF)
- 🧠 Extract skills from resume automatically
- 🎯 Match jobs with your resume using:
  - Skill matching
  - TF-IDF similarity
  - Keyword analysis
- 📈 Shows **match percentage**
- ⚠️ Suggests **missing skills**
- 🔗 Direct **Apply Link** for each job

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Libraries:**  
  - pandas  
  - requests  
  - pdfplumber  
  - scikit-learn  
- **API:** JSearch (RapidAPI)
